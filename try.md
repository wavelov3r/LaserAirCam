### 🛠️ Key Features

| 📷 Vision & UI | 🔌 Connectivity & Control |
| :--- | :--- |
| 🖼️ **Smart Stream:** MJPEG, HD Snapshots, and **StreamHD** (timed high-res captures). | 📶 **Wi-Fi Bridge:** Rock-solid serial link via `ser2net` with smart command queuing. |
| 💻 **WebUI:** Full browser dashboard for camera, LED, and system management. | 💨 **AirAssist:** 24V PWM control, automatic (power-based) or manual override. |
| 🌈 **RGB Status:** Dynamic visual feedback (`running`, `error`, `idle`) via addressable LEDs. | 🏠 **MQTT/HA:** Native Home Assistant discovery for sensors and automation. |

### 📦 Hardware & Design
* 🔘 **Physical Interaction:** 3x hardware buttons for quick local controls.
* 🧩 **Custom Enclosure:** 3D-printable case designed for Atomstack A1 & similar machines.

---

### 🏁 Quick Start
1. Prepare your Raspberry Pi and follow the [**setup.md**](./setup.md) guide.
2. Connect LightBurn via device IP and Virtual COM port.
